# Prompts — Le Istruzioni per l'AI

**Cartella:** `knowledge-core/prompts/`  
**Scopo:** Raccogliere i prompt validati per ogni tipo di task affidato all'AI

---

## A cosa serve questa cartella

Un prompt è la differenza tra un output mediocre e un output eccellente dall'AI. I prompt in questa cartella sono stati **testati, raffinati e approvati** per produrre risultati che rispettano gli standard dell'azienda.

Invece di riscrivere le istruzioni ogni volta, si usa il prompt già validato per quel tipo di task.

---

## Formato di ogni prompt

```markdown
# Prompt: [Nome Task]

**Versione:** 1.x  
**Validato il:** AAAA-MM-GG  
**Output atteso:** Descrizione breve

## Prompt

[Il testo del prompt da copiare e usare]

## Note d'uso

Indicazioni su come personalizzare il prompt per varianti specifiche.

## Esempi di output

Link o riferimento a un output prodotto con questo prompt e approvato.
```

---

## Regola fondamentale

Un prompt entra in questa cartella **solo dopo essere stato testato e aver prodotto output approvati**. I prompt sperimentali rimangono nella sessione di lavoro, non qui.
