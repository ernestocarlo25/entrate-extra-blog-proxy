# Dashboard — I Numeri che Contano

**Cartella:** `knowledge-core/dashboard/`  
**Scopo:** Tenere traccia delle metriche di crescita dell'azienda nel tempo

---

## A cosa serve questa cartella

Senza numeri, non si può sapere se si sta crescendo o decrescendo. La Dashboard è il cruscotto dell'azienda: raccoglie le metriche chiave aggiornate nel tempo, così da avere sempre una visione chiara dello stato del business.

---

## Metriche da tracciare

### Crescita audience

| Metrica | Frequenza aggiornamento |
|---|---|
| Iscritti newsletter totali | Mensile |
| Nuovi iscritti nel mese | Mensile |
| Tasso di apertura newsletter | Per ogni invio |
| Tasso di click newsletter | Per ogni invio |

### Blog e contenuti

| Metrica | Frequenza aggiornamento |
|---|---|
| Articoli pubblicati (totale) | Mensile |
| Visite mensili al blog | Mensile |
| Articolo più letto del mese | Mensile |

### Newsletter "L'Osservatorio"

| Metrica | Frequenza aggiornamento |
|---|---|
| Puntate inviate (totale) | Per ogni invio |
| Destinatari per puntata | Per ogni invio |

---

## Struttura dei file

```
dashboard/
├── 2026-Q3.md    ← numeri luglio-settembre 2026
├── 2026-Q4.md    ← numeri ottobre-dicembre 2026
└── ...
```

Ogni file trimestrale raccoglie le snapshot mensili dello stesso periodo.

---

## Regola di aggiornamento

La Dashboard viene aggiornata **una volta al mese**, ogni primo del mese. Responsabile: CEO.

Un dato non aggiornato è peggio di nessun dato — crea false certezze. Se non si ha tempo di aggiornare, meglio indicare chiaramente "dato non disponibile" piuttosto che lasciare il numero vecchio.
