# Templates — I Modelli Riutilizzabili

**Cartella:** `knowledge-core/templates/`  
**Scopo:** Raccogliere i modelli standard per ogni tipo di documento o contenuto

---

## A cosa serve questa cartella

Un template è un punto di partenza che rispetta già tutti gli standard. Invece di ricominciare da zero ogni volta, si parte dal template corretto per il tipo di contenuto che si sta producendo.

I template eliminano il lavoro ridondante e garantiscono coerenza tra tutti gli output dell'azienda.

---

## Template da creare

| File | Uso |
|---|---|
| `template-articolo-blog.md` | Struttura base per ogni nuovo articolo |
| `template-newsletter.md` | Struttura delle sezioni de "L'Osservatorio" |
| `template-session-report.md` | Formato standard per i report di sessione |
| `template-adr.md` | Formato standard per le decisioni strategiche (ADR) |
| `template-business-unit.md` | Struttura da copiare per ogni nuova BU |
| `template-processo.md` | Formato standard per documentare un processo |

---

## Regola d'uso

Quando si inizia un nuovo contenuto o documento, **cercare prima il template** in questa cartella. Se il template esiste, usarlo come base. Se non esiste, crearne uno dopo aver completato il documento, così il lavoro fatto diventa riutilizzabile in futuro.
